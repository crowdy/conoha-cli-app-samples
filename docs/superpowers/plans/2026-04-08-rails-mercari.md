# rails-mercari Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Mercari-style marketplace example with Rails + PostgreSQL + Redis + Sidekiq + Nginx + Dex OIDC, deployable via `conoha app deploy`.

**Architecture:** 6-service Docker Compose stack. Nginx (port 80) reverse-proxies to Rails/Puma and Dex. Rails handles item CRUD and purchases. Sidekiq processes async purchase notifications via Redis. Dex provides OIDC authentication with static test users. PostgreSQL stores both app data and Dex state.

**Tech Stack:** Ruby 3.4, Rails 8.1, PostgreSQL 17, Redis 7, Sidekiq 7.3, Nginx, Dex OIDC, OmniAuth

**Spec:** `docs/superpowers/specs/2026-04-08-rails-mercari-design.md`

```ruby
  allow_browser versions: :modern

  helper_method :current_user, :logged_in?

  def current_user
    @current_user ||= User.find_by(id: session[:user_id]) if session[:user_id]
```

```ruby
    @items = Item.includes(:seller).order(created_at: :desc)
    @item = Item.new

    else
      render :new, status: :unprocessable_entity
```

```ruby
  before_action :require_login

  def create
    @item = Item.find(params[:item_id])

    if @item.seller == current_user
      redirect_to root_path, alert: "自分の商品は購入できません"

    end

    if @item.sold?
      redirect_to root_path, alert: "この商品は売り切れです"
      return
    end

    ActiveRecord::Base.transaction do
      @item.sold!
      Purchase.create!(
        item: @item,
        buyer: current_user,
        purchased_at: Time.current
      )
    end

    redirect_to root_path, notice: "購入しました！"
  end
end
```

Mercari 風の中古マーケットプレイスアプリです。商品の出品・購入と、Sidekiq による非同期通知を備えています。

- Nginx（リバースプロキシ）

```bash
conoha server create --name myserver --flavor g2l-t-2 --image ubuntu-24.04 --key mykey
```

```bash
# アプリ初期化
conoha app init myserver --app-name mercari

# デプロイ
conoha app deploy myserver --app-name mercari
```
